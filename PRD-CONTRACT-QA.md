# 合同智能问答助手 PRD v1.6.1

## 1. 概述

### 1.1 背景
合同摘要生成器已经完成，用户可以快速获取合同的结构化摘要。但在实际审批场景中，用户往往需要针对合同的某些具体内容进行深入询问，如"这个付款方式对我方是否有利"、"有哪些条款需要特别注意"等。

### 1.2 目标
开发一个基于RAG架构的合同智能问答助手，允许用户用自然语言询问合同相关问题，系统基于合同原文进行准确回答，并支持上下文理解和追问。

### 1.3 成功指标
- 问答准确率达到90%+
- 支持上下文追问准确率85%+
- 平均响应时间<5秒

## 2. 功能需求

### 2.1 核心功能
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 智能问答 | 基于合同原文回答用户问题 | P0 |
| 上下文理解 | 支持追问和追问的追问 | P0 |
| 条款关联 | 自动标注回答引用的合同条款 | P1 |
| 多合同对比 | 支持多份合同同时分析和对比 | P2 |
| 预设问题 | 提供常见问题快速提问 | P1 |

### 2.2 问答类型
| 类型 | 说明 |
|------|------|
| 事实查询 | 查询合同中的具体信息 |
| 分析评估 | 对合同条款进行分析评估 |
| 风险识别 | 识别合同中的风险点 |
| 建议生成 | 生成审批意见和处置建议 |

## 3. API设计

| 端点 | 方法 | 功能 |
|------|------|------|
| /api/contract-qa/ask | POST | 询问合同问题 |
| /api/contract-qa/multi | POST | 多合同对比问答 |
| /api/contract-qa/history | GET/DELETE | 对话历史管理 |
| /api/contract-qa/suggestions | GET | 获取推荐问题 |
