## 2. 技术选型

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| AI对话引擎 | Coze平台 | 意图识别、对话管理 |
| 后端框架 | Express.js | Webhook处理、API服务 |
| 部署平台 | code.coze.cn | Node.js运行环境 |

## 3. 核心模块设计

### 3.1 Webhook处理器
```javascript
POST /webhook/coze - 接收Coze回调，处理用户消息
```

### 3.2 状态查询服务
```javascript
- getPurchaseRequestStatus(id) - 查询采购申请状态
- getPurchaseOrderStatus(id) - 查询采购订单状态
- getGoodsReceipts(params) - 查询收货记录
```

### 3.3 对话意图识别
| 意图 | 处理逻辑 |
|------|----------|
| query_pr_status | 查询采购申请状态 |
| query_po_status | 查询采购订单状态 |
| query_policy | 回答审批政策 |
| general_question | FAQ问答 |

## 4. 数据流设计

```
用户输入: "PR12345到哪了"
    ↓
Coze意图识别: query_pr_status
    ↓
提取实体: { pr_id: "12345" }
    ↓
Webhook回调到后端服务
    ↓
调用 Ariba API: GET /api/purchase-requests/12345
    ↓
解析响应，格式化状态信息
    ↓
返回自然语言响应给用户
```
