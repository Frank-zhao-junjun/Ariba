## 3. 实现内容

### 3.1 创建的文件

| 文件 | 用途 |
|------|------|
| `shared/interfaces/__init__.py` | 接口模块入口 |
| `shared/interfaces/base.py` | 基础接口定义 |
| `shared/interfaces/registry.py` | 模块注册表 |
| `shared/interfaces/factories.py` | 模块工厂 |
| `shared/interfaces/adapters.py` | 适配器系统 |
| `shared/interfaces/examples.py` | 使用示例 |

### 3.2 核心组件

#### BaseModule - 模块基类
```python
class BaseModule(ModuleInterface):
    def initialize(self, config) -> bool
    def shutdown(self) -> bool
    def health_check(self) -> bool
    def get_metadata(self) -> ModuleMetadata
```

#### ModuleRegistry - 注册表
```python
registry = ModuleRegistry()
registry.register("module_name", ModuleClass, dependencies)
registry.get("module_name")
registry.initialize_all()
```

#### ModuleFactory - 工厂
```python
config = ModuleConfig(name="...", class_path="module:Class")
factory = ModuleFactory()
factory.register_from_config(config)
```

### 3.3 依赖注入机制

```python
# 1. 注册模块及其依赖
registry.register(
    "checklist.generator",
    ChecklistModule,
    dependencies=["troubleshooting.query_engine"]
)

# 2. 系统自动注入依赖
checklist = registry.get("checklist.generator")
# checklist.get_dependency("troubleshooting.query_engine") 自动可用

# 3. 延迟加载
config = ModuleConfig(name="...", lazy=True)
```

## 4. 测试验证

### 4.1 接口一致性测试
```python
def test_module_interface():
    # 所有模块实现 BaseModule
    for module in modules:
        assert hasattr(module, 'initialize')
        assert hasattr(module, 'shutdown')
        assert hasattr(module, 'health_check')
```

### 4.2 依赖注入测试
```python
def test_dependency_injection():
    registry = ModuleRegistry()
    registry.register("dep", MockDep)
    registry.register("main", MainModule, dependencies=["dep"])
    
    main = registry.get("main")
    assert main.get_dependency("dep") is not None
```

### 4.3 延迟加载测试
```python
def test_lazy_loading():
    config = ModuleConfig(name="lazy", lazy=True)
    registry.register_lazy(config)
    
    # 尚未加载
    assert "lazy" not in registry.list_instances()
    
    # 触发加载
    registry.get("lazy")
    
    # 已加载
    assert "lazy" in registry.list_instances()
```
