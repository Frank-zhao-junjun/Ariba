## 6. 快速开始

### 6.1 本地开发

```bash
# 启动后端
cd Ariba实施助手
python main.py

# 启动前端
cd web/frontend
npm run dev
```

### 6.2 Docker部署

```bash
# 开发环境
docker-compose -f docker-compose.dev.yml up -d

# 生产环境
./deploy.sh deploy
```

### 6.3 Kubernetes部署

```bash
kubectl apply -f k8s/
```
