## 使用方式

### 在其他模块中使用

```python
# 一键导入所有共享工具
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).parent.parent / "Ariba实施助手"))
from shared_imports import *

# 或直接使用shared模块
from Ariba实施助手.shared import (
    get_logger,        # 获取Logger
    handle_exceptions, # 异常处理装饰器
    safe_execute,      # 安全执行
    ChecklistError,    # 自定义异常
    KnowledgeItem,     # 类型定义
    normalize_version, # 工具函数
)
```

### 日志使用示例

```python
logger = get_logger(__name__)
logger.info("这是一条日志")
```

### 异常处理示例

```python
@handle_exceptions(default_return={}, log_traceback=True)
def risky_function():
    # 可能抛出异常的代码
    pass
```

### 路径使用示例

```python
from shared_imports import get_path, add_to_sys_path

# 获取路径
troubleshooting_path = get_path("troubleshooting")

# 添加到sys.path
add_to_sys_path("checklist")
```
