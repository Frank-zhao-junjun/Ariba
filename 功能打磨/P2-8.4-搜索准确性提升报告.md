## 1. 搜索优化分析

### 1.1 当前搜索问题

| 问题 | 描述 | 影响 |
|------|------|------|
| 匹配单一 | 仅标题匹配 | 召回率低 |
| 无权重 | 字段平等对待 | 准确率低 |
| 无排序 | 顺序随机 | 用户体验差 |
| 无建议 | 无纠错/补全 | 零结果多 |

### 1.2 优化目标

| 指标 | 目标 |
|------|------|
| 精确率 | >80% |
| 召回率 | >90% |
| 平均延迟 | <50ms |
| 零结果率 | <5% |

## 4. 使用示例

```python
from shared.knowledge import SearchOptimizer

# 创建搜索优化器
optimizer = SearchOptimizer(
    min_query_length=2,
    max_results=20,
    fuzzy_threshold=0.6,
    boost_weights={
        "title": 2.0,
        "description": 1.0,
        "solution": 1.5,
        "tags": 1.5,
        "category": 0.5
    }
)

# 执行搜索
result = optimizer.search("登录失败", items)

print(f"找到 {result.total_hits} 个结果")
print(f"耗时 {result.query_time_ms:.2f}ms")
print(f"建议: {result.suggestions}")
```
