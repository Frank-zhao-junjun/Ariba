## 1. 监控需求分析

### 1.1 监控目标

| 类别 | 监控项 | 目标 |
|------|--------|------|
| 系统 | CPU、内存、磁盘 | 资源使用率 < 80% |
| 应用 | 请求延迟、错误率 | 延迟 < 500ms, 错误率 < 1% |
| 业务 | 查询成功率、清单生成 | 成功率 > 99% |
| 基础设施 | 数据库连接、缓存命中率 | 命中率 > 80% |

### 1.2 告警需求

| 级别 | 触发条件 | 通知方式 |
|------|----------|----------|
| INFO | 常规信息事件 | 日志 |
| WARNING | 性能接近阈值 | 日志+Webhook |
| ERROR | 功能异常 | 日志+Webhook+邮件 |
| CRITICAL | 系统不可用 | 全渠道通知 |

## 2. 实现内容

### 2.1 创建的文件

| 文件 | 用途 |
|------|------|
| `shared/monitoring/__init__.py` | 监控模块入口 |
| `shared/monitoring/alerts.py` | 告警系统 |

### 2.2 核心组件

#### MetricsCollector - 指标采集器
```python
collector = get_metrics_collector()
collector.counter("requests_total", 1, {"endpoint": "/api/search"})
collector.gauge("active_users", 100)
collector.histogram("request_duration", 0.5)
```

#### HealthChecker - 健康检查器
```python
checker = get_health_checker()
checker.register_check("database", db_health_check)
result = checker.check_all()
```

#### AlertManager - 告警管理器
```python
manager = get_alert_manager()
manager.add_rule(create_cpu_alert())
manager.add_handler(log_handler)
alerts = manager.evaluate(metrics)
```

### 2.3 指标类型

| 类型 | 用途 | 示例 |
|------|------|------|
| Counter | 累加值 | 请求总数 |
| Gauge | 当前值 | 在线人数 |
| Histogram | 分布统计 | 延迟分布 |

### 2.4 预置告警规则

```python
create_cpu_alert(threshold=90)
create_memory_alert(threshold=90)
create_error_rate_alert(threshold=10)
create_threshold_rule("custom", "metric_key", 100, ">=")
```

## 4. 告警规则示例

### 4.1 系统资源告警

```python
# CPU超过90%告警
AlertRule(
    name="high_cpu",
    condition=lambda m: m.get("cpu") > 90,
    level=AlertLevel.WARNING
)

# 内存超过85%告警
AlertRule(
    name="high_memory",
    condition=lambda m: m.get("memory") > 85,
    level=AlertLevel.ERROR
)
```

### 4.2 应用告警

```python
# 响应时间超过2秒
AlertRule(
    name="slow_response",
    condition=lambda m: m.get("p95_latency") > 2000,
    level=AlertLevel.WARNING
)

# 错误率超过5%
AlertRule(
    name="high_error_rate",
    condition=lambda m: m.get("error_rate") > 5,
    level=AlertLevel.ERROR
)
```

## 5. 通知渠道

### 5.1 支持的渠道

| 渠道 | 用途 | 配置 |
|------|------|------|
| 日志 | 全部告警 | 内置 |
| Webhook | 异步通知 | URL配置 |
| 邮件 | 重要告警 | SMTP配置 |
| 飞书 | 实时通知 | Webhook URL |

### 5.2 处理器使用

```python
# 日志处理器
manager.add_handler(AlertHandlers.log_handler)

# Webhook处理器
webhook_handler = AlertHandlers.webhook_handler("https://example.com/alerts")
manager.add_handler(webhook_handler)
```
