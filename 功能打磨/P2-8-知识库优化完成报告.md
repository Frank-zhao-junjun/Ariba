## 使用示例

### 完整优化流程

```python
from shared.knowledge import (
    KnowledgeDeduplicator,
    KnowledgeLinker,
    VersionManager,
    SearchOptimizer
)

# 1. 知识去重
deduplicator = KnowledgeDeduplicator()
dedup_result = deduplicator.deduplicate(items)
clean_items = dedup_result.merged_items

# 2. 关联优化
linker = KnowledgeLinker()
link_result = linker.optimize_links(clean_items)
optimized_items = linker.apply_suggestions(clean_items, link_result.suggested_links)

# 3. 版本完善
version_mgr = VersionManager()
analysis = version_mgr.analyze_versions(optimized_items)
suggestions = version_mgr.suggest_missing_versions(optimized_items)
final_items = version_mgr.apply_version_suggestions(optimized_items, suggestions)

# 4. 搜索测试
optimizer = SearchOptimizer()
result = optimizer.search("登录失败", final_items)
print(f"找到 {result.total_hits} 个结果")
```
