## 1. 去重分析

### 1.1 发现的重复问题

| 问题类型 | 描述 | 影响 |
|----------|------|------|
| 精确重复 | 标题和内容完全相同 | 数据冗余 |
| 近似重复 | 标题相似，内容略有差异 | 维护困难 |
| 版本差异 | 同一问题多版本重复 | 搜索混乱 |

### 1.2 去重策略

| 策略 | 阈值 | 适用场景 |
|------|------|----------|
| 精确匹配 | 100% | 标题完全相同 |
| 标题相似 | ≥85% | 问题标题近似 |
| 内容相似 | ≥75% | 描述/解决方案重叠 |

## 3. 核心功能

### 3.1 精确匹配去重
- 基于标题生成签名
- 哈希比较
- O(n) 时间复杂度

### 3.2 相似度去重
- 标题相似度 (SequenceMatcher)
- 内容相似度 (TF-IDF)
- 二元组匹配

### 3.3 智能合并
- 保留最完整项
- 合并关联数据
- 保持引用完整性

## 4. 使用示例

```python
from shared.knowledge import KnowledgeDeduplicator

# 创建去重器
deduplicator = KnowledgeDeduplicator(
    strategy=DeduplicationStrategy.SIMILAR_TITLE,
    title_threshold=0.85,
    content_threshold=0.75
)

# 执行去重
result = deduplicator.deduplicate(items)

print(f"发现 {len(result.duplicate_groups)} 组重复")
print(f"移除 {result.removed_count} 个重复项")
```
