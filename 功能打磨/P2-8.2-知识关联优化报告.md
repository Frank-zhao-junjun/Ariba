## 1. 关联分析

### 1.1 关联类型

| 关联类型 | 说明 | 置信度 |
|----------|------|--------|
| category | 同类别关联 | 0.8 |
| tag | 共同标签 | 0.5-0.95 |
| solution | 解决方案相关 | 0.75 |
| semantic | 语义相关 | 0.6-0.9 |

### 1.2 发现的关联问题

| 问题 | 描述 | 影响 |
|------|------|------|
| 孤立项 | 无关联的知识项 | 难以发现 |
| 链接缺失 | 相关项未关联 | 知识孤岛 |
| 过度关联 | 关联过多 | 维护困难 |

## 4. 使用示例

    min_link_confidence=0.6,
    enable_semantic_link=True,
    max_links_per_item=10
)

# 优化链接

print(f"已有链接: {result.existing_links}")
print(f"新增建议: {result.suggested_links}")
print(f"孤立项: {len(result.orphaned_items)}")
