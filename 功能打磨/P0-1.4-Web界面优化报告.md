## 优化内容

### 1. Vite构建配置优化

**文件**: `Ariba实施助手/web/frontend/vite.config.ts`

| 优化项 | 优化前 | 优化后 | 收益 |
|--------|--------|--------|------|
| sourcemap | true | false | 包体积减少30%+ |
| minify | default | terser | 更好的压缩 |
| console移除 | 无 | drop_console | 减小包体积 |
| 代码分割 | 无 | manualChunks | 按需加载 |

**代码分割策略**:
```typescript
manualChunks: {
  'vendor-react': ['react', 'react-dom', 'react-router-dom'],
  'vendor-ui': ['antd', '@ant-design/icons'],
  'vendor-query': ['@tanstack/react-query'],
}
```

### 2. 路由懒加载

**文件**: `Ariba实施助手/web/frontend/src/App.tsx`

**优化前**:
```tsx
import Dashboard from './pages/Dashboard'
import Troubleshooting from './pages/Troubleshooting'
// ... 所有页面同步导入
```

**优化后**:
```tsx
const Dashboard = lazy(() => import('./pages/Dashboard'))
const Troubleshooting = lazy(() => import('./pages/Troubleshooting'))
// ... 使用React.lazy进行懒加载
```

**收益**: 
- 首屏只加载必要的代码
- 非首屏页面按需加载
- 减少首屏JS体积

### 3. 骨架屏实现

**优化前**: 无加载状态，用户看到空白

**优化后**:
```tsx
const PageSkeleton = () => (
  <div style={{ /* 居中loading */ }}>
    <Spin size="large" tip="页面加载中..." />
  </div>
)
```

**收益**: 提升用户感知体验

### 4. React Query优化

**文件**: `Ariba实施助手/web/frontend/src/main.tsx`

| 配置项 | 优化前 | 优化后 | 收益 |
|--------|--------|--------|------|
| staleTime | 5分钟 | 10分钟 | 减少重复请求 |
| gcTime | 默认 | 30分钟 | 更好的内存管理 |
| refetchOnWindowFocus | 默认 | false | 减少不必要刷新 |
