# Ariba助手 - 智能审批状态查询功能 PRD

## 1. 概述

### 1.1 产品目标
为Ariba采购系统用户提供基于自然语言的智能审批状态查询助手，让用户能7x24小时通过对话方式快速获取采购申请、订单、发票的审批进度。

### 1.2 核心价值
- **减少支持工单**: 自动化回答80%以上的状态查询请求
- **提升用户体验**: 自然语言交互，无需记忆系统操作路径
- **提高效率**: 用户自助查询，释放采购人员精力

## 2. 用户故事

### US-001: 用户查询采购申请状态
**作为** 申请员工
**我想要** 通过对话快速知道我的采购申请审批到哪一步了
**以便于** 及时跟进审批进度，合理安排工作

### US-002: 用户查询采购订单状态
**作为** 需求部门负责人
**我想要** 查询我部门提交的PO审批状态
**以便于** 了解供应商何时能收到订单

### US-003: 用户询问审批政策
**作为** 新员工
**我想要** 了解采购审批的政策和流程
**以便于** 正确提交采购申请

### US-004: 审批超时提醒
**作为** 系统
**我想要** 在审批超时前主动提醒审批人
**以便于** 加快审批流程

## 3. 功能需求

| 功能编号 | 功能名称 | 优先级 | 描述 |
|----------|----------|--------|------|
| F001 | 自然语言理解 | P0 | 解析用户查询意图，提取实体（PR号、PO号、时间等） |
| F002 | 审批状态查询 | P0 | 调用Ariba API获取审批状态 |
| F003 | 历史审批记录 | P0 | 展示审批节点历史 |
| F004 | 审批政策问答 | P1 | FAQ式回答审批政策问题 |
| F005 | 审批超时监控 | P2 | 监控并提醒超时审批 |

## 4. 技术方案

### 4.1 技术栈
- **后端**: Node.js + Express
- **AI**: Coze平台（对话机器人）
- **数据源**: Ariba API
- **部署**: code.coze.cn
