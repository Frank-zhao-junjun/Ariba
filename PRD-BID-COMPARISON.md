# PRD-BID-COMPARISON.md - AI询价比价助手

## 1. 概述

### 1.1 背景
当前Ariba助手已具备RFx/RFP自动生成功能，但在收到供应商报价后，用户仍需手动整理比价表、计算权重得分、分析最优供应商。这项工作重复性高、耗时长，采购人员迫切需要智能化支持。

### 1.2 目标
开发AI询价比价助手，自动读取多个供应商的报价文件，生成结构化比价分析表，包含：
- 价格对比矩阵
- 技术评分（基于评分标准）
- 综合排名与推荐
- 风险提示与建议

### 1.3 成功指标
- 比价报告生成时间 < 30秒
- 供应商排名准确率 95%+
- 支持PDF/Word/Excel多格式报价自动解析

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 报价文件上传 | 支持PDF/Word/Excel多格式批量上传 | P0 |
| 智能解析 | 自动提取供应商名称、物料、数量、单价、交期等关键信息 | P0 |
| 比价矩阵生成 | 生成标准比价分析表（Comparison Matrix） | P0 |
| 权重评分 | 根据预设权重自动计算综合得分 | P1 |
| AI推荐 | 基于价格+质量+交期+供应商评级综合推荐最优供应商 | P1 |
| 风险提示 | 识别价格异常、资质缺失等风险点 | P2 |
| 导出报告 | 支持导出Excel/PDF格式的比价报告 | P1 |

### 2.2 技术方案

#### 报价解析策略
| 格式 | 解析方式 |
|------|----------|
| PDF | pdf-parse + 正则匹配关键字段 |
| Excel | xlsx库直接读取 |
| Word | mammoth库提取文本 |
