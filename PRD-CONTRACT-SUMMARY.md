# 智能合同摘要生成器 PRD v1.6

## 1. 概述

### 1.1 背景
SAP Ariba实施顾问在日常工作中需要频繁审查和分析各类采购合同，传统方式依赖人工阅读，效率低下且容易遗漏关键信息。

### 1.2 目标
开发一个基于AI的合同摘要生成器，自动分析合同文本，提取关键条款，识别潜在风险，并生成结构化摘要。

### 1.3 成功指标
- 合同审查时间缩短80%
- 风险识别准确率达到95%+
- 用户满意度≥4.5/5

## 2. 功能需求

### 2.1 核心功能
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 一键摘要 | 自动生成50字以内的合同一句话摘要 | P0 |
| 条款提取 | 自动识别并提取15+标准合同条款 | P0 |
| 风险检测 | 识别金额/时间/质量/法律/合规风险 | P0 |
| 合规检查 | 验证合同是否符合企业采购政策 | P1 |
| 智能建议 | 针对识别的问题提供改进建议 | P1 |

### 2.2 条款类型
- 合同当事人
- 合同金额与付款
- 交付与验收
- 质量保证
- 违约责任
- 保密条款
- 知识产权
- 争议解决
- 合同期限
- 不可抗力
- 续约与终止
- 其他重要条款

### 2.3 风险等级
| 等级 | 说明 | 颜色标识 |
|------|------|----------|
| 高风险 | 需要立即关注和修改 | 🔴 红色 |
| 中风险 | 建议优化 | 🟡 黄色 |
| 低风险 | 可接受但建议注意 | 🟢 绿色 |

## 3. 技术方案

### 3.1 架构
```
前端(HTML) → API Layer → Contract Summary Service → LLM Service
```

### 3.2 API设计
| 端点 | 方法 | 功能 |
|------|------|------|
| `/api/contract-summary/generate` | POST | 生成完整摘要 |
| `/api/contract-summary/brief` | POST | 一句话摘要 |
| `/api/contract-summary/clauses` | POST | 提取条款 |
| `/api/contract-summary/risks` | POST | 风险检测 |
| `/api/contract-summary/demo` | GET | 演示数据 |

### 3.3 技术栈
- Node.js + Express
- Coze API / OpenAI
- REST API
