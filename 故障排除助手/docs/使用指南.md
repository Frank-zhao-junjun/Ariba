# 故障排除助手使用指南

> 版本：1.0.0

## 快速开始

### 安装

```bash
# 确保Python 3.8+
python --version

# 安装依赖
pip install pydantic
```

### 基本使用

```python
import json
from core.query_engine import create_query_engine

# 加载知识数据
with open("fixtures/sample_knowledge.json", "r") as f:
    knowledge_data = json.load(f)

# 创建查询引擎
engine = create_query_engine(knowledge_data)

# 执行查询
results = engine.search("供应商登录失败")
```

## API参考

### QueryEngine.search()

**参数**:
- `query` (str): 搜索关键词
- `version_tags` (List[str]): 版本过滤标签
- `limit` (int): 返回结果数量，默认10
- `min_score` (float): 最低匹配分数，默认0

**返回**: `List[QueryResult]`

### QueryResult

- `item`: KnowledgeItem 对象
- `score`: 匹配分数 (0-100)
- `match_type`: 匹配类型
- `matched_keywords`: 匹配的关键词

## 版本标签

| 标签 | 说明 |
|------|------|
| #V2505 | 2025年5月版本 |
| #V2602 | 2026年2月版本 |
| #V2604 | 2026年4月版本 |
| #V2605 | 2026年5月版本 |
| #VNextGen | Next Generation版本 |
| #VClassic | Classic版本 |

## 示例查询

```python
# 示例1: 基本搜索
results = engine.search("审批卡住")

# 示例2: 带版本过滤
results = engine.search(
    "发票匹配",
    version_tags=["#V2605"]
)

# 示例3: 限制结果数量
results = engine.search("供应商", limit=5)

# 示例4: 设置最低分数
results = engine.search("登录", min_score=30)
```
