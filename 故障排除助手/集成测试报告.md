## 快速开始

```python
# 基本使用
from core.query_engine import create_query_engine
import json

# 加载知识
with open("fixtures/sample_knowledge.json") as f:
    data = json.load(f)

# 创建引擎
engine = create_query_engine(data)

# 搜索
results = engine.search("供应商登录失败", version_tags=["#V2605"])
for r in results:
    print(f"[{r.score}] {r.item.title}")
```
