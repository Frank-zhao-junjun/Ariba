# PRD-PR-Faq-Assistant v1.9

## 1. 概念与愿景

**采购申请FAQ智能助手** - 嵌入式AI助手，帮助采购申请人在填写申请时即时获取政策、流程、操作问题的答案，减少采购部门80%的支持工单。

**用户感受**: 像有个"懂行的同事"在旁边随时解答问题，不用打断别人工作，不用翻手册，不用问完还不好意思。

---

## 2. 功能范围

### 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **政策问答** | 回答采购政策、金额限制、审批权限等问题 | P0 |
| **流程指引** | 指导如何选择供应商、填写申请、提交审批 | P0 |
| **系统操作** | 解答如何使用Ariba系统的具体操作 | P1 |
| **问题分类** | 自动识别问题类型并路由到对应知识库 | P1 |
| **多轮对话** | 支持追问，上下文记忆 | P2 |

### 问答场景示例

**政策类**:
- "超过多少钱需要总经理审批？"
- "哪些品类必须使用优选供应商？"
- "没有供应商资质可以采购吗？"

**流程类**:
- "采购IT设备走什么流程？"
- "如何添加新的供应商？"
- "审批到哪一步了？"

**操作类**:
- "如何上传附件？"
- "在哪里查看历史申请？"
- "怎么修改已提交的申请？"

## 3. 技术方案

### API设计

```
POST /api/pr-faq/ask
Request: { "question": "string", "context": object }
Response: { "answer": "string", "category": "policy|process|operation", "confidence": number }

POST /api/pr-faq/categories
Response: { "categories": ["policy", "process", "operation", "system"] }

GET /api/pr-faq/popular
Response: { "questions": ["高频问题列表"] }
```

### 知识库结构

```javascript
{
  category: "policy",
  question: "超过多少钱需要总经理审批",
  answer: "根据公司采购政策：\n- 10万以下：部门经理审批\n- 10-50万：总监审批\n- 50万以上：总经理审批",
  keywords: ["审批", "金额", "总经理", "权限"]
}
```

## 4. 验收标准

| 标准 | 描述 |
|------|------|
| FAQ-001 | 用户提问后3秒内返回答案 |
| FAQ-002 | 政策类问题准确率≥90% |
| FAQ-003 | 支持至少20个常见问题的智能回答 |
| FAQ-004 | 答案包含来源说明和操作指引 |

---

## 5. 非目标

- 不做复杂的自然语言理解（当前版本基于关键词匹配）
- 不做多语言支持（当前仅支持中文）
- 不做人工客服转接（当前版本纯自动化）
