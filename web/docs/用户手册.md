## 系统概述

Ariba实施助手是一款面向SAP Ariba实施项目的辅助工具，提供故障知识查询、检查清单生成、项目管理等功能。

### 主要功能

- 🔍 **故障排除助手**: 快速查询SAP Ariba常见问题解决方案
- ✅ **清单管理**: 管理实施项目检查清单
- 📊 **统计分析**: 项目完成情况可视化
- 🕸️ **知识图谱**: 故障知识关联可视化

## 仪表盘

仪表盘提供系统概览，包括：

- **知识库统计**: 知识总数、版本分布
- **项目状态**: 进行中项目、平均完成率
- **系统健康度**: 0-100评分
- **最近更新**: 最新知识条目
- **告警提示**: 需要关注的事项

## 故障排除助手

### 查询故障知识

1. 在搜索框输入问题关键词
2. 可选择版本和模块进行筛选
3. 点击查询按钮获取结果

### 查看详情

1. 点击结果卡片查看详情
2. 包含问题描述、解决方案、相关文档
3. 可提交反馈帮助改进

### 筛选功能

- **版本筛选**: 按SAP Ariba版本筛选
- **模块筛选**: 按功能模块筛选
- **匹配度排序**: 按相关度排序结果

## 清单管理

### 查看项目列表

1. 进入清单管理页面
2. 查看所有项目及其进度
3. 点击项目名称查看详情

### 更新清单项

1. 进入项目详情页
2. 按阶段查看清单项
3. 勾选复选框标记完成
4. 系统自动更新进度

### 项目状态

| 状态 | 说明 |
|------|------|
| 进行中 | 项目正在实施中 |
| 已完成 | 所有清单项已完成 |
| 已暂停 | 项目暂停实施 |

## 创建新项目

### 步骤1: 项目信息

填写项目名称和描述。

### 步骤2: 企业信息

- 选择企业规模
- 选择所属行业
- 选择现有系统
- 选择集成级别

### 步骤3: 模块选择

选择需要实施的模块：

- 寻源管理 (Sourcing)
- 合同管理 (Contract)
- 采购到付款 (Buying)
- 供应商管理 (Supplier)
- 支出分析 (Spending)

### 步骤4: 版本与确认

- 选择目标版本
- 确认项目配置
- 点击创建完成

## 知识图谱

### 功能特点

- 节点类型: 错误、原因、解决方案、模块
- 交互功能: 拖拽、缩放、点击查看详情
- 版本筛选: 按版本过滤图谱数据

### 操作说明

- **拖拽**: 拖动节点调整布局
- **缩放**: 使用鼠标滚轮缩放
- **全屏**: 点击全屏按钮查看
- **筛选**: 按版本筛选图谱

### 节点颜色

| 颜色 | 类型 |
|------|------|
| 红色 | 错误 |
| 黄色 | 原因 |
| 绿色 | 解决方案 |
| 蓝色 | 模块 |

## 常见问题

### Q: 如何导出清单数据？

A: 当前版本暂不支持导出功能，后续版本将支持Excel/CSV导出。

### Q: 如何添加项目成员？

A: 当前版本为单人使用，后续版本将支持多用户协作。

### Q: 数据是否自动保存？

A: 是的，所有操作都会实时保存到服务器。
