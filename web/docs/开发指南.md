# Ariba实施助手 开发指南

## 技术栈

### 前端
- **框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **UI组件库**: Ant Design 5
- **图表库**: ECharts 5
- **状态管理**: Zustand + React Query
- **路由**: React Router v6
- **HTTP客户端**: Axios

### 后端
- **框架**: FastAPI
- **服务器**: Uvicorn
- **数据验证**: Pydantic v2
- **文档**: Swagger/OpenAPI

## 开发规范

### TypeScript规范

1. 使用strict模式
2. 定义完整类型，避免使用any
3. 组件使用FC泛型
4. 导出类型定义到types/index.ts

### React规范

1. 使用函数组件 + Hooks
2. 组件放在components目录
3. 页面放在pages目录
4. 使用CSS Modules或内联样式

### API规范

1. 统一BaseResponse格式
2. 使用Pydantic进行数据验证
3. 所有路由添加tags
4. 提供完整的API文档

### 命名规范

- 组件: PascalCase (如Dashboard.tsx)
- 函数/变量: camelCase
- 常量: UPPER_SNAKE_CASE
- 文件: kebab-case

## 开发流程

### 1. 安装依赖

```bash
# 后端
cd backend
pip install -r requirements.txt

# 前端
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
# 后端
python api_server.py

# 前端 (新终端)
npm run dev
```

### 3. 添加新页面

1. 在 `pages/` 创建页面组件
2. 在 `App.tsx` 添加路由
3. 在 `Layout` 添加菜单项

### 4. 添加新API

1. 在 `models/schemas.py` 定义请求/响应模型
2. 在 `api/` 创建路由文件
3. 在 `api_server.py` 注册路由

### 5. 添加新组件

1. 在 `components/` 创建组件
2. 导出到对应index.ts
3. 在页面中导入使用
